# Clone and Try Locally
Clone Repository
```
git clone https://github.com/CareerLattice/careerlatice-web-app.git
```

Go to Repository
```
cd careerlatice-web-app
```

Install Vendor 
```
composer install
```

Generate App Key
```
php artisan key:generate
```

Generate env files then Run the program
```
php artisan serve
```
